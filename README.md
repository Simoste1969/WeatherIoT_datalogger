# WeatherIoT Datalogger  
### Professional IoT Weather & Environmental Data Logger  
**Reference Demo Project for WeatherIoT – Klinkon Electronics**
info@klinkonelectronics.com

## Overview

**WeatherIoT Datalogger** is an **official demonstration project** designed to showcase the capabilities of the **WeatherIoT board**, engineered and manufactured by **Klinkon Electronics**, combined with the **BME680 environmental sensor**.

It serves as a **reference design**, **application demo**, and **commercial development base** for:

- IoT weather data loggers  
- Professional weather stations  
- Advanced environmental monitoring systems  
- Industrial and scientific IoT solutions  

---

## Why WeatherIoT

The **WeatherIoT** platform is specifically designed for **environmental and meteorological monitoring**, integrating all essential components into a single, compact, and IoT-ready board.

### Key features demonstrated
- Native support for digital environmental sensors  
- Integrated **microSD slot** for local data logging  
- Integrated **RTC** for accurate timestamps  
- Onboard **status LED**  
- Architecture optimized for IoT and remote sensing applications  

This project demonstrates how to build a complete system **without external hardware**, minimizing complexity and development effort.

---

## BME680 – Advanced environmental sensing

The project uses the **BME680**, a high-performance environmental sensor capable of measuring:

- Temperature  
- Relative humidity  
- Atmospheric pressure  
- Air quality (VOC / gas)  

This makes the **WeatherIoT + BME680** combination ideal for build a prototype of a professional environmental monitoring, research, smart agriculture, smart cities or industrial application.

---

## Status LED – Immediate feedback

The **integrated status LED** provides visual feedback for:

- system startup;
- correct sensor operation;
- SD card or runtime errors.

An essential feature for standalone and remote deployments.

---

## Integrated microSD data logging

The **onboard microSD** enables:

- continuous environmental data logging;
- reliable long-term data storage;
- easy offline analysis and data integration.

---

## Integrated RTC – Accurate timestamps

The **Real Time Clock (RTC)** ensures:

- precise timestamps for each data sample;
- network-independent operation;
- time continuity during power outages.
